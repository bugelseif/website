Primeiros passos com Python RPA

Por Bruna Ferreira

Em Videos

tags: Python RPA

Participei do meetup da BotCity "Os desafios no desenvolvimento de projetos RPA" apresentando os primeiros passos com Python RPA, em formato de livecoding. Desenvolvemos uma automação simples, a qual copia o nome de arquivos .txt que representam notas fiscais e os insere em um arquivo .csv. No final deixei um desafio para quem quiser praticar modificando o código como desejar, o código inicial está disponível nesse repositório.
